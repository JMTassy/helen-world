# HELEN OS — The Governed World

**Imagination is free. The map obeys admitted moves.**

A living demo of a governed, non-sovereign AI substrate: an AI can propose, dream, and explain
anything — but only an **operator-admitted receipt** changes reality. Everything else stays a
candidate, forever honest about being one.

- **index.html** — *The Governed World.* Feel the firewall: propose anything → it appears as a muted
  candidate → admit it as operator → it becomes a gold receipt that lights the world.
- **chromatic.html** — *The Chromatic Architecture.* Governance made legible by colour **and glyph**
  (◇ open = proposing, ◆ sealed = gating) — the 6-epoch flow from Violet Nebula to Gold Admission.
- **knowledge-territories.html** — *Dream of Conquest · Knowledge Territories.* Answer trials → earn
  ZOL → claim territories around the Kernel Castle.
